Index Page

Page Information

  • Main

    Main page provides registration and login for both tutors and students.
    Additionally, provides information on subject offerings and the tutoring environment.

  • Student Register

    Registers students by inserting information into Students table.
    Once registerd, students will be directed to account page.

  • Tutor Register

    Registers tutors by inserting information into Tutors table.
    Once registerd, tutors will be directed to lessons page.

  • Login

    • Allows students and tutors to login to view account information.
    Requires declaration of type of account to determine which
    table (Students or Tutors) will be used to verify username
    and password.

  • Account (Students)

    After login, studentID will be used to find and list all
    scheduled appointments in StudentsClasses. These appointments
    can be updated or deleted as well. Clicking on date allows
    student to make a new appointment that will be inserted into
    TimeSlots and eventually Classes once a tutor signs up to teach
    the class.

  • Lessons (Tutors)

    After login, tutorID will be used to find and display all
    scheduled appointments in TutorsClasses. These appoinments can be cancelled which
    will remove the tutor from this class listing in Classes. Tutors will
    have access to view available appointments by viewing a listing
    of appointments found in StudentClasses but not found in Classes. Once tutor agrees
    to teach this classe, all information will be added to Classes.

  • Admin

    Admin page provides administration the ability to view, search, and sort the Students,
    Tutors, Subjects, and TimeSlots tables along with the option to delete the account.
    Additionally, the admin page provides the ability to insert, edit, and delete Subjects
    and TimeSlots. Lastly, the admin page has the ability to alter the Classes table
    by selecting a different tutor for upcoming StudentsClasses.

    Admin login as tutor with username and password provided in project report..
